\section{Core Package (\texttt{src/core/})}
\label{sec:core}
% ─────────────────────────────────────────────────────────────────────────────

The core package provides shared primitive types, configuration loading, CLI argument
parsing, and the seeded random-number generator used throughout the simulation.

\subsection{\texttt{Vec2} (struct)}
\label{sec:vec2}

Plain aggregate for 2-D floating-point coordinates.  Chosen as a struct (no encapsulation
overhead) because it appears in tight physics loops.

\texttt{AgentId} is a type alias: \texttt{using AgentId = std::uint32\_t}.  It uniquely
identifies each agent within a generation.

% ─────────────────────────────────────────────────────────────────────────────
\subsection{\texttt{BoundaryMode} (enum class)}

\begin{lstlisting}
enum class BoundaryMode { Clamp, Wrap };
\end{lstlisting}

\texttt{Clamp} constrains positions to $[0,w]\times[0,h]$.
\texttt{Wrap} applies toroidal topology via \texttt{std::fmod}.
Controlled by the \texttt{"boundary\_mode"} field in \texttt{default\_config.json}.

\subsection{\texttt{Random} (class)}

Single 64-bit Mersenne Twister instance shared by simulation, evolution, and data layers.
A single instance per simulation ensures reproducible runs from any given seed.

\begin{table}[H]
\centering\small
\caption{\texttt{Random} interface}
\begin{tabularx}{\textwidth}{@{}lX@{}}
\toprule
\textbf{Method} & \textbf{Description} \\
\midrule
\texttt{Random(seed: uint64\_t)} & Construct; seed the MT19937-64 engine \\
\texttt{next\_int(min, max)} & Uniform integer in $[\mathit{min}, \mathit{max}]$ \\
\texttt{next\_float(min, max)} & Uniform float in $[\mathit{min}, \mathit{max}]$ \\
\texttt{next\_gaussian(mean, stddev)} & Normal distribution sample \\
\texttt{next\_bool(probability)} & True with given probability $p\in[0,1]$ \\
\texttt{weighted\_select(weights)} & Index proportional to weight values \\
\texttt{shuffle<T>(vec)} & Fisher-Yates in-place shuffle \\
\texttt{sample\_indices(total, count)} & Sample \emph{count} distinct indices from $[0,\mathit{total})$ \\
\texttt{seed()} \texttt{const} & Return the construction seed \\
\texttt{engine()} & Direct access for seeding from checkpoints \\
\bottomrule
\end{tabularx}
\end{table}

\textbf{Private members:}
\begin{itemize}[nosep]
  \item \texttt{seed\_: uint64\_t} — stored to support checkpoint save/restore.
  \item \texttt{engine\_: std::mt19937\_64} — the underlying generator.
\end{itemize}
